Preguntas de examen

1. ¿Qué es Node.js?

Respuesta: Node.js es un entorno de ejecución (runtime) que permite ejecutar JavaScript fuera del navegador.

2. ¿Qué es un stack?

Respuesta: Un stack es el conjunto de tecnologías que utilizas para desarrollar una aplicación o un proyecto.

3. ¿Qué diferencia hay entre Frontend y Backend?

  • Frontend: se ejecuta en el navegador y gestiona la interfaz de usuario.
  • Backend: se ejecuta en el servidor y gestiona la lógica, los datos y las APIs.
